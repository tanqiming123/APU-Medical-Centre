# APU Medical Centre Management System

A comprehensive Java application designed to streamline medical center operations. This project implements a **Role-Based Access Control (RBAC)** system to ensure that different users have access to specific functionalities based on their responsibilities.

## 2. Features & Roles

The system is divided into four primary modules:

* **Manager:** Oversees system operations, manages staff records, and views center-wide reports.
* **Staff:** Handles administrative tasks, manages patient registrations, and schedules appointments.
* **Doctor:** Accesses patient medical history, records consultation notes, and manages personal schedules.
* **Customer (Patient):** Views available doctors, books/cancels appointments, and checks personal medical records.
